# How to use the screenshot inbox (non-technical)

## Goal
Use screenshots as a fallback input for restaurants whose menus are hard to fetch automatically, like Facebook-only menus.

## Where to upload
Root folder:
<https://drive.google.com/drive/folders/1vvEfxJdk-nbIiWmOuE1WGXApdidGMCyT>

Current live inbox:
- John Bull Pub inbox:
  <https://drive.google.com/drive/folders/1z4yW70fpxQCGHV8wYWtL-ycS8wWV2Gla>

## What to do
1. Save the screenshot from Facebook or another source
2. Open the correct restaurant `inbox` folder
3. Upload the image there
4. Prefer a filename like:
   - `2026-06-03-menu.jpg`
   - or `2026-06-03-1.jpg`
5. Tell me you uploaded it, and I can process it

## What happens after that
- I can scan the inbox folder
- download the image
- read the screenshot
- turn it into structured menu rows
- put it through the review/build workflow

## Important rule
Only upload screenshots to the matching restaurant folder.
Do not mix different restaurants in the same inbox.
