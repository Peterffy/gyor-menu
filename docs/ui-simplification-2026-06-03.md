# UI simplification update — 2026-06-03

## Main changes requested
The public UI was simplified to prioritize faster access to menus.

## Changes made
1. Main title changed to:
   - `Mi a menü?`
2. Replaced `Ma / Holnap` tabs with weekday tabs:
   - Hétfő
   - Kedd
   - Szerda
   - Csütörtök
   - Péntek
3. Current weekday is the default active tab.
4. Past weekdays are visually greyed.
5. Removed price display from menu cards and detail pages.
6. Removed the public reliability legend section.
7. Removed public search, filter, sort, and share controls.
8. Page order is now:
   - weekday tabs
   - favorite places
   - menu list
9. Restaurant detail pages were updated to match the weekday-tab navigation model.

## Product effect
The page is now more focused on one job:
- quickly seeing what is on the menu on a chosen weekday

with less UI overhead before the user reaches the menu list.
